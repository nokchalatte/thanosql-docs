# ThanoSQL Docs

ThanoSQL 기술 문서 - https://docs.thanosql.ai

## Run development server

### Prerequisites

- [pipenv](https://github.com/pypa/pipenv)

### Setup

```bash
pipenv install
```

### Local Run

```bash
pipenv shell
mkdocs serve
```
